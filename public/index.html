API Pública - Clima & Lua de Rio Verde-GO Brasil

Bem-vindo! Esta API fornece acesso gratuito e público a dados meteorológicos e astronômicos para a cidade de Rio Verde, Goiás. Obtenha facilmente informações como temperatura atual, previsão para os próximos dias e fases da lua.

A API é pública e pode ser usada por qualquer pessoa, sem necessidade de chave de autenticação.
Base URL: https://api-weather-rioverde-go.onrender.com/

Como Usar Endpoints Exemplos Testando Notas

Como Usar

  • Faça uma requisição GET para /tempo/rioverde para obter o clima atual.
  • Exemplo: GET https://api-weather-rioverde-go.onrender.com/tempo/rioverde
  • Veja a documentação completa abaixo para mais detalhes.

Limites

  • Sem autenticação, mas com limite de 1000 requisições/dia por IP.
  • Se precisar de mais, entre em contato.

Endpoints

Método Endpoint Descrição
GET /tempo/rioverde Retorna o clima atual de Rio Verde
GET /rioverde/moon Retorna a fase da lua e dados astronômicos atuais
GET /previsao/rioverde/:data Retorna previsão do tempo e dados astronômicos para uma data futura

            Parâmetro
            Tipo
            Descrição

            data
            string
